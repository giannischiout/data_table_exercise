
# EnterSoftOne

React app built with **Vite** and **TypeScript**, using MUI, Redux Toolkit, React Router, and SWR.

---

## Prerequisites

- **Node.js** (LTS recommended)
- **pnpm** â€” install globally: `npm install -g pnpm`

---

## Setup

1. Clone the repo and go into the project folder:
   ```bash
   cd entersoftone
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the dev server:
   ```bash
   pnpm dev
   ```
   The app runs at **http://localhost:5173** (or the port Vite prints in the terminal).

---

## Scripts

| Command          | Description                    |
|------------------|--------------------------------|
| `pnpm dev`       | Start development server       |
| `pnpm build`     | TypeScript check + production build |
| `pnpm preview`   | Preview production build       |
| `pnpm lint`      | Run ESLint                     |
